# ❤️ Ruqia's Love Website

## 📖 How to Access

Your interactive love website is ready! Follow these steps to access it:

### **Option 1: Direct Raw File Access (Works Now!)**
👉 **[Click Here to Open Website](https://raw.githubusercontent.com/sheikhjunaid1329-png/devops/main/index.html)**

*Note: This opens the raw HTML file - copy the URL and open it in your browser.*

### **Option 2: Enable GitHub Pages (Recommended)**

1. Go to your repository settings: https://github.com/sheikhjunaid1329-png/devops/settings
2. Scroll down to **"GitHub Pages"** section
3. Under "Source", select **"Deploy from a branch"**
4. Select branch: **main**
5. Select folder: **/ (root)**
6. Click **Save**

Your site will be available at: `https://sheikhjunaid1329-png.github.io/devops/`
